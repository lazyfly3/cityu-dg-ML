# CityU DG Machine Learning Course Materials / 城大數據科學機器學習課程資料

本倉庫包含香港城市大學數據科學專業機器學習課程的所有知識點總結。

This repository contains all knowledge summaries for the Machine Learning course in the Data Science program at City University of Hong Kong.

## 目錄結構 / Directory Structure

```
.
├── Assignments/              # 作業相關文件 / Assignment files
│   ├── Assignment_*_学习笔记_Study_Notes.md    # 作業學習筆記 / Assignment study notes
│   ├── 数学基础补充_Math_Basics.md            # 數學基礎補充 / Math basics supplement
│   └── 考试知识点总结_Exam_Cheat_Sheet.md     # 考試知識點總結 / Exam cheat sheet
│
└── lec/                      # 課程講義 / Lecture materials
    ├── Lecture1/
    │   └── Lecture1_知识点总结.md
    ├── Lecture2/
    │   └── Lecture2_知识点总结.md
    ├── ...
    └── Lecture12/
        └── Lecture12_知识点总结.md
```

## 內容說明 / Content Description

### 作業知識點總結 / Assignment Knowledge Summaries

- **Assignment 1**: 概率論基礎、朴素貝葉斯、線性判別分析、KL散度
- **Assignment 2**: 泊松分布、梯度下降、約束優化、支持向量機
- **Assignment 3**: 矩陣恒等式、線性系統、坐標下降、軟間隔SVM
- **Assignment 4**: 對稱矩陣、主成分分析、線性系統、DTFT

### 課程講義知識點總結 / Lecture Knowledge Summaries

- **Lecture 1**: 機器學習導論、監督學習基礎、概率論回顧
- **Lecture 2**: 線性回歸、最大似然估計、梯度下降、正則化
- **Lecture 3**: 邏輯回歸、分類問題、決策邊界、多類分類
- **Lecture 4**: 朴素貝葉斯、貝葉斯推斷、生成式vs判別式
- **Lecture 5**: 線性判別分析、二次判別分析、高斯判別分析
- **Lecture 6**: 支持向量機、最大間隔、硬間隔vs軟間隔、核技巧
- **Lecture 7**: 決策樹、分裂準則、剪枝、隨機森林
- **Lecture 8**: 神經網絡導論、感知機、多層感知機、反向傳播
- **Lecture 9**: 主成分分析、降維、特征值分解、應用
- **Lecture 10**: 聚類、K-Means、層次聚類、評估指標
- **Lecture 11**: 集成方法、Bagging、Boosting、Stacking
- **Lecture 12**: 模型評估、交叉驗證、超參數調優、模型選擇

## 格式說明 / Format Description

每個知識點總結文件包含：

1. **目錄 / Table of Contents**: 快速導航
2. **主要知識點 / Main Topics**: 中英文雙語解釋
3. **數學定義 / Mathematical Definitions**: 完整的公式和推導
4. **符號說明 / Symbol Explanations**: 詳細的符號含義
5. **計算步驟 / Calculation Steps**: 逐步計算指南
6. **計算示例 / Calculation Examples**: 具體數值示例
7. **通俗解釋 / Intuitive Explanations**: 易於理解的類比
8. **補充要點 / Additional Details**: 擴展知識
9. **學習建議 / Learning Recommendations**: 學習指南
10. **參考資源 / Reference Resources**: 推薦閱讀
11. **練習題 / Practice Problems**: 練習題目
12. **例題與解答 / Worked Examples**: 詳細解答

## 使用說明 / Usage Instructions

1. 按順序閱讀各Lecture的知識點總結
2. 結合Assignment學習筆記加深理解
3. 完成練習題並參考例題解答
4. 使用考試知識點總結進行復習
