# CityU DG Machine Learning Course Materials / 城大数据科学机器学习课程资料

本仓库包含香港城市大学数据科学专业机器学习课程的所有知识点总结。

This repository contains all knowledge summaries for the Machine Learning course in the Data Science program at City University of Hong Kong.

```

│   ├── 数学基础补充_Math_Basics.md            # 数学基础补充 / Math basics supplement

    ├── Lecture1/
    │   └── Lecture1_知识点总结.md
    ├── Lecture2/
    │   └── Lecture2_知识点总结.md
    ├── ...
    └── Lecture12/
        └── Lecture12_知识点总结.md
```

## 内容说明 / Content Description

### 作业知识点总结 / Assignment Knowledge Summaries

- **Assignment 1**: 概率论基础、朴素贝叶斯、线性判别分析、KL散度
- **Assignment 2**: 泊松分布、梯度下降、约束优化、支持向量机
- **Assignment 3**: 矩阵恒等式、线性系统、坐标下降、软间隔SVM
- **Assignment 4**: 对称矩阵、主成分分析、线性系统、DTFT

### 课程讲义知识点总结 / Lecture Knowledge Summaries

- **Lecture 1**: 机器学习导论、监督学习基础、概率论回顾
- **Lecture 2**: 线性回归、最大似然估计、梯度下降、正则化
- **Lecture 3**: 逻辑回归、分类问题、决策边界、多类分类
- **Lecture 4**: 朴素贝叶斯、贝叶斯推断、生成式vs判别式
- **Lecture 5**: 线性判别分析、二次判别分析、高斯判别分析
- **Lecture 6**: 支持向量机、最大间隔、硬间隔vs软间隔、核技巧
- **Lecture 7**: 决策树、分裂准则、剪枝、随机森林
- **Lecture 8**: 神经网络导论、感知机、多层感知机、反向传播
- **Lecture 9**: 主成分分析、降维、特征值分解、应用
- **Lecture 10**: 聚类、K-Means、层次聚类、评估指标
- **Lecture 11**: 集成方法、Bagging、Boosting、Stacking
- **Lecture 12**: 模型评估、交叉验证、超参数调优、模型选择

## 格式说明 / Format Description

每个知识点总结文件包含：

1. **目录 / Table of Contents**: 快速导航
2. **主要知识点 / Main Topics**: 中英文双语解释
3. **数学定义 / Mathematical Definitions**: 完整的公式和推导
4. **符号说明 / Symbol Explanations**: 详细的符号含义
5. **计算步骤 / Calculation Steps**: 逐步计算指南
6. **计算示例 / Calculation Examples**: 具体数值示例
7. **通俗解释 / Intuitive Explanations**: 易于理解的类比
8. **补充要点 / Additional Details**: 扩展知识
9. **学习建议 / Learning Recommendations**: 学习指南
10. **参考资源 / Reference Resources**: 推荐阅读
11. **练习题 / Practice Problems**: 练习题目
12. **例题与解答 / Worked Examples**: 详细解答

## 使用说明 / Usage Instructions

1. 按顺序阅读各Lecture的知识点总结
2. 结合Assignment学习笔记加深理解
3. 完成练习题并参考例题解答
4. 使用考试知识点总结进行复习
